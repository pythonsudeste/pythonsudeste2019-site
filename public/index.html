Sobre

Conferência

Depois de passar por Belo Horizonte, Rio de Janeiro e São Paulo, a quarta conferência das comunidades Python do Sudeste vai acontecer dessa vez no Espírito Santo. A Python Sudeste reúne centenas de pessoas durante 3 dias para discutir sobre software livre, práticas de desenvolvimento e Python.

Leia nosso Código de Conduta

Organização

Esse ano a Python Sudeste é organizada pela comunidade Python do Espírito Santo, com a ajuda da comunidade de São Paulo, Rio de Janeiro e Minas Gerais.
